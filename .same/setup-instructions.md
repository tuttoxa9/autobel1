# Инструкция по настройке AutoBel

## Что было сделано:

### 1. Интеграция с Firestore
- Страница **контактов** теперь загружает данные из Firestore:
  - Коллекция: `pages`
  - Документ: `contacts`
  - Структура данных: title, subtitle, address, phone, email, workingHours, socialMedia

- Страница **"О нас"** загружает данные из Firestore:
  - Коллекция: `pages`
  - Документ: `about`
  - Структура данных: stats, companyInfo, bankDetails

### 2. Яндекс Карты
- Создан компонент `YandexMap` для отображения карты на странице контактов
- API ключ настраивается через админку в разделе "Настройки" → "Основные"
- Карта автоматически геокодирует адрес и добавляет метку

### 3. Формы обратной связи
- Формы на странице контактов сохраняются в Firestore:
  - Коллекция: `contact-forms`
  - Поля: name, phone, message, timestamp, status

## Как настроить:

### 1. Получить API ключ Яндекс.Карт:
1. Перейти на https://developer.tech.yandex.ru/
2. Зарегистрироваться и создать новый проект
3. Получить API ключ для JavaScript API
4. Добавить домен вашего сайта в настройки ключа

### 2. Настроить API ключ в админке:
1. Перейти в админку сайта: `/adminbel`
2. Открыть раздел "Настройки"
3. Во вкладке "Основные" найти поле "API ключ Яндекс.Карт"
4. Вставить полученный API ключ и сохранить

### 3. Настроить контент страниц через админку:
1. **Контакты**: Раздел "Страницы" → "Контакты"
   - Заполнить адрес, телефон, email
   - Настроить время работы
   - Добавить ссылки на социальные сети

2. **О нас**: Раздел "Страницы" → "О нас"
   - Обновить статистику компании
   - Заполнить реквизиты и банковские данные

### 4. Проверить работу:
- Перейти на страницу `/contacts` - должна отображаться карта с адресом
- Заполнить форму обратной связи - данные должны сохраниться в Firestore
- Проверить, что данные загружаются из админки на обеих страницах
